Final Project Proposal

Topic

This project is a helper for the user to manage shopping experience, by providing a shopping list, a consumption keeper, and an overview of the monthly and weekly shopping.

Audience

People who need to make list before shopping and keep track of their consumptions on shopping.

Design & Style

The design and colors for the website will be clean, simple, and comfortable.

The main colors for this website is blue.

Inspired By:

Coolors Listify Application Mint

Color Palette:

#F1FAEE

#A8DADC

#457B9D

#1D3557

Scope

The website will have three sections for different functions.

  • shopping list (what you need to buy)
  • spend tracking
  • personal overview

Extras:

Event-driven DOM Manipulation in the shopping list section
